# Como funciona

Ao se tornar um **Inscrito** no canal do [parkejunior](https://www.twitch.tv/parkejunior), você desbloqueia uma série de benefícios exclusivos e recebe uma **rolada** na nossa roleta de recompensas.

Este guia explica os detalhes sobre como funcionam as recompensas, como utilizá-las, como trocar ou tranferir itens.

Apoiando o canal com **Inscrição** ou **Incrição de Presente**, enviando **100 bits** ou doando **R\$ 10** no Livepix, você recebe uma rolada na nossa roleta\!

Os tipos de item que você pode receber na roleta são:

## 📌 Como escolher o conteúdo

Quer escolher uma receita para fazer ou o filme para assistir na live? Você pode escolher o conteúdo do item que você ganhou na Roleta.

Siga esses passos:

1. **Verifique seu inventário**
   - Liste seus itens disponíveis com `!roladas`.
2. **Execute o comando de escolha**:

   ```bash
   !roladas escolher <item> <detalhe>
   ```
   - `<item>`: tipo de item a ser escolhido (ex: `filme`, `receita`).
   - `<detalhe>`:
     - **Filmes/Séries/Animes**: cole o link completo do IMDb (ex: `https://www.imdb.com/title/tt0123456/`).
     - **Balões, Receitas e Bricks**: descreva brevemente (ex: `Chapéu de palha com fitas coloridas`).
3. **Confirmação**
   - A **SayuritaBOT** responderá confirmando sua escolha.
   - Seu item será salvo com o conteúdo que você escolheu na lista de recompensas.

<Tip>
  **Dica:** Use o comando `!roladas` após escolher para conferir seu saldo de itens.
</Tip>

## ✏️ Como trocar itens

Tem algum item que você ainda não escolheu e que está parado pegando pó? Você pode trocar itens que você já possui por outros.

Para trocar, siga este passo a passo:

1. **Revise seus itens**
   - Liste seus itens disponíveis com `!roladas`.
2. **Escolha o tipo de troca**:
   - **Troca direta (filme ↔ série)**

     ```bash
     !roladas trocar <item_antigo> = <item_novo>
     ```
     - `<item_antigo>` e `<item_novo>`: apenas `filme` ou `série`.
   - **Combinação de itens (2 → 1)**

     ```bash
     !roladas trocar <item_um> + <item_dois> = <item_novo>
     ```
     - `<item_um>` e `<item_dois>`: `bricks`, `balão` ou `receita`.
     - `<item_novo>`: qualquer tipo (`bricks`, `balão`, `receita`, `filme`, `série`).
3. **Confirmação**
   - A **SayuritaBOT** responderá confirmando a troca.
   - Os itens antigos serão marcados como _"trocado"_ e seu novo item aparecerá na lista de recompensas.

<Tip>
  **Dica:** Verifique com `!roladas` se você realmente possui os itens necessários antes de trocar.
</Tip>

## 🎁 Como transferir um item para outro usuário

Quer presentear alguém com um dos seus itens? Use o comando de transferência:

1. **Identifique o item** que deseja transferir:
   - Liste seus itens disponíveis com `!roladas`.
2. **Execute o comando**:

   ```bash
   !roladas transferir <item> <usuário_destino>
   ```
   - `<item>`: tipo de item a ser transferido (ex: `balão`, `filme`).
   - `<usuário_destino>`: inclua `@` para garantir o registro correto.
3. **Confirmação**
   - A **SayuritaBOT** responderá confirmando a transferência.
   - O seu item será marcado com _"transferido"_ e acrescentado na lista do usuário que recebeu.

<Tip>
  **Dica:** Antes de transferir, confirme o nome do usuário para evitar envios incorretos.
</Tip>

## ✖️ Uso do multiplicador

Os comandos `!roladas add` e `!roladas transferir` aceitam um **multiplicador opcional** para indicar que a ação deve acontecer várias vezes de uma só vez. O multiplicador é um número inteiro seguido da letra `x` (sem espaço), por exemplo: `5x`.

**Sintaxe básica:**

```bash
!roladas add <username> <tipo_de_donate> <item> [multiplicador]x
!roladas transferir <item> <usuário_destino> [multiplicador]x
```

**Exemplos:**

```bash
!roladas add @parkejunior resub filme 5x     # adiciona 5 itens 'filme' para @parkejunior
!roladas transferir receita @whaliaj 3x      # transfere 3 itens 'receita' para @whaliaj
```

<Note>O multiplicador é aplicado à quantidade do **mesmo** item especificado no comando.</Note>

<Tip>**Dica:** Confirme o usuário e o item antes de executar comandos com multiplicador para evitar erros em lote.</Tip>
